Developed and implemented a speech recognition system using Connectionist Temporal Classification objective function based deep bidirectional LSTM RNN model that directly transcribes audio data to text, without requiring an intermediate phonetic representation. Explored models based on combination of RNNs and the CTC objective function (with and without language model) and models based on semi-CRF and RNNs (segmental RNN) for the sequence labeling problem. Studied attention and memory augmented based RNN models for some tasks including machine translation, handwriting synthesis, and speech recognition
